= Blacklight::Folders

== Developer Notes

==== Install Jetty

  $ rake jetty:unzip
  $ rake jetty:add_test_core
  $ rake jetty:start

==== Generate a test app and run the specs

  $ rake jetty:start
  $ rake engine_cart:clean
  $ rake engine_cart:generate
  $ rake spec

==== Run the test app rails server

  $ rake engine_cart:generate
  $ cd spec/internal
  $ rails s

==== Generate data for your test app

  $ cd spec/internal
  $ rake blacklight:solr:seed
