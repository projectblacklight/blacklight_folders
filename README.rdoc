= Blacklight::Folders

== Developer Notes

=== Install Jetty

  $ rake jetty:unzip
  $ rake jetty:start

=== Generate a test app and run the specs

  $ rake engine_cart:clean
  $ rake engine_cart:generate
  $ rake spec

=== Run the test app rails server

  $ rake engine_cart:generate
  $ cd spec/internal
  $ rails s
